Instructions

In the first section, we can search for stocks symbols, even if we do not remember the symbol name. In the second section, the search will be with the full and correct symbol name. In order to look for Microsoft stock, we will use the next symbol: MSFT (can search in lower case as well).
The details are taken from alphavantage. The API allows to conduct up to 2 searches in a minute.

In order to create this website, the following technologies were used:

  • JavaScript
  • Json
  • Visual Studio
  • D3 library
  • Rapid API alpha vantage
